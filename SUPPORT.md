# Support

If you need help with gerbil-data, here are the available channels:

## Documentation

- [README](README.md) — Quick start, architecture overview, and feature reference
- [docs/](docs/) — Detailed specifications (encoding, binary format, dataset format)
- [CONTRIBUTING.md](CONTRIBUTING.md) — Development setup and contribution guidelines

## Community Support

- **GitHub Issues** — [Report bugs](https://github.com/shardzhang/gerbil-data/issues/new?template=bug_report.md) or [request features](https://github.com/shardzhang/gerbil-data/issues/new?template=feature_request.md)
- **Discussions** — Use GitHub Discussions for questions, ideas, and general help

## Professional Support

For commercial support or consulting, please contact the maintainers directly.

## Before Asking

1. Check the [existing issues](https://github.com/shardzhang/gerbil-data/issues) — your question may already be answered
2. Review the documentation in the [docs/](docs/) directory
3. Ensure you have met all [prerequisites](README.md#prerequisites)
